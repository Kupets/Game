Игра "Тенис"
==========================

Для запуска и сборки необходима установленная JAVA версии 1.7 или выше. Для сборки необходим Maven версии 3.2.

Что бы собрать проект Maven'ом выполнить:
```
mvn package
```

Что бы запустить проект из корня выполнить:
```
java -jar tenis.jar
```
